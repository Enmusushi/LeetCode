##### 数组
1.数组是一个引用数据类型.
2.什么是数组：一组排列有序的数据：在内存当中，开辟的**连续**的内存空间
- 声明：在变量类型或者变量名后面加上[]
- 类型：可以是任意数据类型（基本数据类型，引用数据类型）
	1.每一种数据类型其默认值：
		a.基本数据类型：
			int:0
			char：空格
			double:0.0
			boolean:false
		b.引用数据类型：
			String :null
- 名称：参照变量名的命名
- 值：使用关键字new，new的目的是通知虚拟机，在堆中开辟内存空间；
	给定数据类型，告诉虚拟机，我们需要开辟的内存空间，是要保存什么样的数据类型；
	给定数组长度，告诉虚拟机，我们需要开辟多少个数据空间；
- 声明并创建数组
	1.声明
	2.new 一个新的数组
- 数组的数据顺序，是依靠游标来确定的，数组的游标是从0开始的
